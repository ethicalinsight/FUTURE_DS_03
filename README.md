**Track Code:** DS

---

## 🔄 Workflow

2. **Preprocessing (Excel):**

3. **Dashboard Development (Power BI):**

   - Compared channel performance and conversion efficiency

- Generated actionable recommendations to improve lead-to-customer conversion rates.
